Built for high performance

Advanced screenshot workflows reimagined for modern professionals.

DPI & Multi-Monitor Support

Instantly tracks your cursor across multiple screens, adapting screen layouts and resolutions perfectly for crisp, sharp cropping overlays on each screen.

Pixel Grid Magnifier

Get pixel-perfect coordinate previews. Use the mouse scroll wheel to adjust zoom size in real time for precise cropping bounds.

Lightweight & Fast

Engineered using native GPU hooks and lightweight scripting. Starts up in milliseconds and operates silently in your system tray.

Advanced Image Editor

Includes speech bubbles, highlighters, geometric shapes, blur overlays, drawing tools, and Paint-style canvas resize handles to alter your workspace.

Drag and Drop Support

Drag photos directly from Windows Explorer into the editor

Professional Annotations

Highlight code segments, draw diagrams, explain user flows, and blur out sensitive credentials in seconds. Feathershot replaces expensive screenshot tools with a beautiful, zero-bloat editor.

  • Autosave: Save screenshots to custom folders based on timestamp pattern templates.
  • Tray Minimization: Automatically hides inside tray menu to keep your taskbar clean.
  • Direct Clipboard Copying: Hit a checkbox to skip the editor entirely.

Power-user Shortcuts

Fly through tasks with quick global hotkeys and built-in hotkey customization.

Shortcut Action
PrtSc or Ctrl + Shift + S Trigger region screenshot
Ctrl + C Copy crop to Clipboard (inside Editor)
Ctrl + S Save crop directly to file
Ctrl + Z / Ctrl + Y Undo / Redo steps
Esc Cancel screenshot or close window
